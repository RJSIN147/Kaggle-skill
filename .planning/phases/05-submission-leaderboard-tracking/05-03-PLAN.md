## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Kaggle API -> submissions_log parse | `description`, `status`, `date`, `publicScore` are Kaggle-authored, i.e. UNTRUSTED text |
| kaggle CLI stdout/stderr -> our process | The raw buffer may carry a token-shaped string |
| submissions_log -> control/submissions.jsonl | A crash mid-write must not corrupt the canonical record |

Pinned by tests/test_no_credential_leak.py. |

| T-05-03-SC | Tampering | npm/pip/cargo installs | accept | This plan installs nothing (zero new dependencies). |
