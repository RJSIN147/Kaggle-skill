## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| credential store (env / ~/.kaggle) -> checker | secret read into process memory; must never be printed |
| checker -> Kaggle API | live authenticated call; only exit code consumed |
| kaggle CLI availability -> checker | missing CLI must degrade to UNVALIDATED, not crash |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-02 | Information Disclosure | any print/echo of a credential | mitigate | `_mask` helper + env-var-name-only output; `tests/test_no_credential_leak.py` regex gate over scripts/ |
| T-01-03 | Information Disclosure | world/group-readable kaggle.json | mitigate | `_ensure_mode_600` self-heal, consent-gated (D-06a) |
| T-01-05 | Tampering | silent kaggle install | mitigate | `shutil.which` guard + consent-gated `uv pip install kaggle`; never auto-install (D-07/D-03) |
| T-01-10 | Spoofing | false-pass on failed auth | mitigate | Decide strictly by exit code of `kaggle competitions list`; capture real 401 strings; never infer success from stdout text |
</threat_model>
