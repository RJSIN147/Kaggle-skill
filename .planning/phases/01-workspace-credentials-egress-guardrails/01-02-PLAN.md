Safe-merge (D-02) — create-if-absent for files; deep-merge (add missing keys, keep existing values) for control-plane JSON:
```python
def create_if_absent(path, content):  # returns "create" | "skip"
    if path.exists(): return "skip"
    path.parent.mkdir(parents=True, exist_ok=True); path.write_text(content); return "create"
```
