## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| workspace → api.kaggle.com | The egress allowlist decides whether the flow's traffic leaves; Claude proves it live |
| exit code → SKILL human loop | The SKILL is the only waiter; a mis-documented protocol could reintroduce a busy-loop |
| Kaggle rules gate → human | Rules acceptance is UI-only (no API); an auto-approved verify would falsely mark it done |
| control/raw provenance → git | Operator staging must be explicit-path so the gitignored last-error.txt is never swept in |
| live transcript → output | Real Kaggle calls could leak a token-shaped string |
