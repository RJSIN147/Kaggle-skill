<objective>
Close Gap 2 (review WR-01) from `02-VERIFICATION.md`: `download_data.py` (lines 167-179) routes
EVERY non-zero return from the download call — including rc=127 (kaggle CLI missing) and rc=124
(timeout), both RESERVED by `kaggle_gateway.run_kaggle` — into the UI-gate `classify_gate` branch,
printing a misleading "clear whichever UI-only gate applies in a browser" instruction + exit 77 when
the real remediation is "install the CLI" / "check egress". The SKILL then tells the human to accept
the rules → re-run → CLI still missing → 77 again → an unresolvable loop.

tailored messages. This plan mirrors that branch logic into `download_data.py` BEFORE the `if rc != 0`
UI-gate fallthrough — without reintroducing any busy-loop (no sleep/poll).

Purpose: make COMP-02's "one clear browser instruction" guarantee hold in the missing-CLI / timeout
precondition instead of degrading to a misleading rules-acceptance instruction.
Output: a missing CLI (127) / stalled egress (124) gets its own correct remediation and exit code;
the exit-77 UI-gate path is reserved for real 403 gates.
</objective>
